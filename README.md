# Family Clean Quest

Android-приложение для семейной уборки: личные баллы, командный счет, награды и просмотр выполненных дел другими участниками без права редактирования.

## Запуск

```powershell
npm install
npm run start
```

Затем в Expo выберите запуск на Android-устройстве или эмуляторе.

## Синхронизация

В `app.json` замените Firebase-поля в `extra` на настройки своего проекта Firebase. После этого приложение сможет использовать общий Firestore-документ семьи.

## Правила доступа

- Активный пользователь может отмечать свои выполненные дела.
- Все участники видят выполненные дела друг друга.
- Дети не могут удалять или корректировать чужие выполненные дела.
- Профиль `Мама` сейчас считается родительским и может убрать ошибочную запись из журнала.

## Что уже заложено

- Участники: Мама, Вероника 14 лет, Диана 12 лет.
- Личные баллы и общий командный счет.
- Задания по комнатам, ванной, лестнице, прихожей, огороду, грядкам и цветникам.
- Полив стоит 1 балл, прополка стоит 2 балла.
- Личные и командные награды со списанием баллов.
- Локальное хранение на телефоне и подготовленный слой Firebase/Firestore.
