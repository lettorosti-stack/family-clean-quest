# Family Clean Quest

Android-приложение для семейной уборки: личные баллы, командный счет, награды и просмотр выполненных дел другими участниками без права редактирования.

## Запуск

```powershell
npm install
npm run start
```

Затем в Expo выберите запуск на Android-устройстве или эмуляторе.

## Синхронизация

Приложение использует Firebase Anonymous Authentication и Cloud Firestore. Публичная web-конфигурация проекта хранится в `app.json`, а правила доступа — в `firestore.rules`.

На первом телефоне мама открывает настройки профиля и нажимает **Создать семью**. Приложение покажет 12-значный код. На остальных телефонах нужно выбрать **Подключиться по коду** и ввести этот код.

Синхронизируются:

- выполненные задачи и баллы;
- покупки, голосования и статусы наград;
- пользовательские задачи;
- аватары и запросы сброса пароля;
- удаления записей и изменения с нескольких устройств.

Пароли профилей и кодовое слово мамы остаются только на конкретном устройстве и не отправляются в Firestore открытым текстом.

Перед сборкой необходимо включить Anonymous Authentication, создать Firestore и опубликовать `firestore.rules`.

## Правила доступа

- Активный пользователь может отмечать свои выполненные дела.
- Все участники видят выполненные дела друг друга.
- Дети не могут удалять или корректировать чужие выполненные дела.
- Профиль `Мама` сейчас считается родительским и может убрать ошибочную запись из журнала.
