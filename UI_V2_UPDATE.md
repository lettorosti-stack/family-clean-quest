# Family Clean Quest 2.0 — обновление интерфейса

Обновление собрано поверх текущей версии 1.2.0 и не меняет ключи существующих данных.

## Сохранено без обнуления
- completed — история выполненных задач и начисленные баллы;
- purchases — запросы и полученные награды;
- customTasks — пользовательские задачи;
- family code и Firebase-синхронизация;
- аватары и настройки пользователей.

## Изменено
- новый мобильный визуальный стиль;
- нижняя навигация: Сегодня, Дом, Дела, Награды, Семья;
- отдельный экран «Дом» с полным списком зон;
- сохранены гараж/автомобиль и помощь бабушке и дедушке;
- экран «Семья» с личными и командными баллами и недавней историей;
- уведомления сохранены и доступны через колокольчик;
- иконки встраиваются в src/previewHtml.js при генерации.

## Проверено
- inline JavaScript preview.html;
- WebView bridge и authoritative sync;
- генерация src/previewHtml.js с встроенными изображениями.

Перед сборкой APK выполните `npm.cmd install`, `npx.cmd expo-doctor`, затем запустите GitHub Actions.
